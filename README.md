# Simple pong in C ver. 0.01)
C project made as a part of first week of study.

## Libaries:

* <curses.h>
* <stdio.h>
* <stdlib.h>
* <time.h>


## To compile pong:

* Use GCC: `gcc -Wall -Werror -Wextra pong.c -lncurses`
* Use Makeile by simply: `make`

## Controls:

### To exit:

* `q`

### Left player
* up/down `a` / `z`

### Right player
* up/down `k` / `m`
